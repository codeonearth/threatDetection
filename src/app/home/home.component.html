Faq

What SIEM tool?

SIEM (Security Information and Event Management) tools are software solutions that collect, aggregate, and analyze security-related data from various sources in real-time to detect threats, manage incidents, and ensure compliance. They consolidate logs, alerts, and other data from network devices, servers, applications, and other security systems into a central location for analysis, enabling security teams to identify malicious activity and respond to potential breaches.

What are mitre att&ck?

MITRE ATT&CK is a global knowledge base of adversarial tactics and techniques based on real-world observations of cyberattacks. It provides a structured framework for cybersecurity professionals to understand, detect, and defend against threats by categorizing attacker behaviors into "tactics" (high-level objectives) and "techniques" (specific methods used to achieve those objectives). This common language helps security teams build better defenses, test their security posture, and respond more effectively to incidents.
